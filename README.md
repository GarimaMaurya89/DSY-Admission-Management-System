# Direct Second Year Diploma Admission Management System

## Overview
The Direct Second Year Diploma Admission Management System is a Python-based application designed to automate and simplify the admission process for direct second-year diploma students. The system reduces manual work and improves accuracy, efficiency, and transparency in admission management.

## Features
- Student registration and login
- Admin login and dashboard
- Student data management
- Merit list generation
- PDF generation for admission-related documents
- User-friendly and modern GUI using CustomTkinter

## Technologies Used
- Python
- CustomTkinter (GUI)
- MySQL (Database)
- PyMySQL (Database connectivity)
- ReportLab (PDF generation)

## How to Run the Project
1. Clone the repository
2. Install required libraries:
3. Set up MySQL database locally.
4. Update database credentials in the code as per your local configuration.
5. Run the main Python file to start the application.

## Database Note
Database credentials and actual student data are not included in this repository for security reasons. The application uses a local MySQL database setup.
